Uma Jornada de Aprendizado

O jogo From The Abyss nasceu como parte de um projeto de TCC no curso de Desenvolvimento de Sistemas na Etec. Mesmo com desafios, limitações e diversas vezes refazendo todo o TCC, foi desenvolvido com dedicação, aprendizado e muita paixão.

Agradecemos à:

  • Professores pelo Feedback e Paciência
  • Comunidade da Unity e de Animadores
  • Todos que nos acompanharam durante esse último ano

Tecnologias Utilizadas:

Unity C# HTML/CSS JavaScript Photopea Figma Pixilart Aseprite Itch.io ElevenLabs Netlify Git/GitHub RemoveBG Visual Studio Code Audacity Google Fonts Font Awesome Cursor
